## Setup
1. Run the **setup.sh** script or equivalent commands to the ones below
´´´
	bower install -d
	npm install -d
	grunt concat
	grunt uglify
´´´
2. Run **run_client.sh** script.
3. Run **run_server.sh** script.
4. Go to localhost:8088 and have fun.
